# 使用说明书（实操版）

本页是“怎么点、怎么填、看到什么算成功”的操作手册。

## 0. 使用前确认

### 前置条件

- 已执行 `pnpm install`
- 已配置 `.env` 与 `.env.server`
- 已执行 `pnpm dev:all`
- `http://localhost:8787/health` 返回 `ok: true`

---

## 1. 聊天功能（`/chat`）

### 操作步骤

1. 打开左侧导航 **AI 聊天**。
2. 在输入框输入问题，例如：`帮我设计一个 React 登录页`。
3. 点击发送或按 Enter。
4. 观察回答是否逐段流式出现。
5. 点击复制按钮验证内容可复制。

### 成功判定

- 能看到模型回答持续追加，而非一次性全部出现。
- 能复制消息文本。

### 常见问题

- **没有流式效果**：确认后端已启动，且 Key 有效。
- **请求报错**：检查 `.env.server` 中 `TONGYI_API_KEY`。

---

## 2. 代码生成功能（`/codegen`）

### 操作步骤（自然语言场景）

1. 打开 **代码生成** 页面。
2. 场景选择 `natural`。
3. 输入需求，例如：`写一个带表单校验的注册页`。
4. 点击生成。
5. 等待流式输出完成后，复制代码块。

### 操作步骤（截图场景）

1. 场景切换为 `screenshot`。
2. 上传 UI 截图（PNG/JPG）。
3. 可选填写补充描述（布局、主题、交互）。
4. 点击生成并查看输出。

### 成功判定

- 页面持续输出代码文本。
- 生成结果包含可读的组件结构与样式。

### 常见问题

- **截图模式无结果**：确认 `VISION_MODEL` 配置正确。
- **返回权限错误**：检查视觉模型是否有调用权限。

---

## 3. 在线编辑器（`/editor`）

### 操作步骤

1. 打开 **在线编辑器** 页面。
2. 新建或切换文件标签。
3. 输入一段 TS/JS 代码，观察语法提示。
4. 执行格式化（Format Document）。
5. 创建快照并尝试恢复。

### 成功判定

- 可多文件切换编辑。
- 能看到诊断提示与格式化效果。
- 快照可保存并恢复。

---

## 4. RAG 知识库问答（`/rag`）

### 操作步骤

1. 打开 **知识库检索** 页面。
2. 点击上传，选择 `.txt` 或 `.md` 文档。
3. 点击 **建库**。
4. 建库成功后，在提问框输入问题。
5. 点击 **检索并回答**。

### 成功判定

- 页面显示“已建库，xx 片段”。
- 回答可以流式输出，并与文档内容相关。

### 常见问题

- **建库失败**：文件过大、过多或内容为空。
- **回答不准**：文档内容不足，建议补充更完整资料后重建。

---

## 5. 图表功能（`/charts`）

### 操作步骤

1. 打开 **可视化** 页面。
2. 输入自然语言描述，例如：`生成近 7 天销售额折线图`。
3. 点击生成图表。
4. 检查图例、坐标轴、数据项是否合理。

### 成功判定

- 能渲染出 ECharts 图形。
- 修改描述后图表会跟随变化。

---

## 6. 调试修复（`/debug`）

### 操作步骤

1. 打开 **调试修复** 页面。
2. 粘贴报错信息或问题代码。
3. 触发分析。
4. 查看修复建议与优化建议。

### 成功判定

- 返回内容包含“问题定位 + 修复方向”。
- 能直接据此改代码验证。

---

## 7. 多模态（`/multimodal`）

### 操作步骤

1. 打开 **多模态** 页面。
2. 上传图片（UI 截图、报错截图等）。
3. 输入你想分析的问题。
4. 提交并查看结果。

### 成功判定

- 返回内容能结合图片信息进行描述或建议。

---

## 8. 个人中心（`/profile`）

### 操作步骤

1. 打开 **个人中心**。
2. 检查模型配置与偏好项。
3. 根据需要开启或关闭路由守卫。

### 成功判定

- 设置保存后刷新页面仍生效。

---

## 9. 5 分钟演示流程（推荐）

1. `/` 展示工作台指标。
2. `/chat` 发起一次流式问答。
3. `/codegen` 生成一个组件。
4. `/editor` 展示格式化 + 快照恢复。
5. `/rag` 上传文档并问答。
6. `/charts` 生成图表结束演示。

这套流程可以完整覆盖“问答 -> 生成 -> 编辑 -> 检索 -> 可视化”。
